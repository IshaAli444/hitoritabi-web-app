# hitoritabi-web-app
Full-stack web application built independently and deployed on Render.

Hitoritabi is a web-based application designed for writing, reading, solo travel planning and journaling. 
This project was built independently from concept to deployment as a full-stack web application.


🔗 Live Demo: https://hitoritabi.onrender.com/

---

##  Features

- Responsive user interface
- Clean and structured project architecture
- Dynamic content rendering
- Backend integration 
- Deployment using Render

---

##  Tech Stack

Frontend:
- HTML
- CSS tailwind
- JavaScript

Backend:
- [Node.js / Express ]

Database:
- [PostgreSQL]

Deployment:
- Render
